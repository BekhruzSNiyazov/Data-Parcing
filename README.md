# Data parsing
This is a program that does data parsing. It was written in C.
# Usage
- Compile the program
- Run
```bash
./main data.ini section.key
```
`data.ini` file should look similar to this:
```ini
section1
key1 = value1
key2 = value2
key3 = value3

section2
key4 = value4
key5 = value5
key6 = value6

section3
key7 = value7
key8 = value8
key9 = value9
```
